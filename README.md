# 🏷️ سوق النطاقات

موقع بسيط لعرض النطاقات المتاحة للبيع، يعمل على GitHub Pages.

## 🚀 المميزات
- عرض النطاقات كبطاقات أنيقة
- بحث بالاسم والوصف
- تصفية حسب التصنيف (تقنية، طبية، تعليمية، مالية، عقارات)
- تصفية حسب السعر (من - إلى)
- عرض حالة النطاق (متاح / محجوز / مباع)
- متجاوب مع جميع الأجهزة

## 📁 هيكل الملفات
- `index.html` - الصفحة الرئيسية
- `style.css` - التصميم
- `script.js` - المنطق البرمجي
- `domains.json` - قاعدة البيانات

## 🔧 كيفية التحديث
1. عدّل ملف `domains.json` لإضافة أو حذف أو تعديل النطاقات
2. ارفع التغييرات إلى GitHub
3. الموقع يتحدث تلقائياً
